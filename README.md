## Запуск ##

После скачивания проекта необходимо выполнить несколько шагов,
чтобы запустить приложение. Сначала нужно запустить 
контейнер командой: 

`docker compose up -d --build`

Перейти внутрь контейнера:

`docker exec -it php bash`

Инициализировать проект:

`php init`

В процессе инициализации надо будет ответить на несколько 
вопросов. Окружение можно выбрать любое, а вот на предложение
перезаписать файл `yii` надо ответить отказом (там правка, 
необходимая для чтения `.env`).

Затем необходимо подтянуть зависимости через композер:

`composer install`

Выполнить миграции:

`php yii migrate`

Запустить слушатель очереди:

`php yii queue/listen`

После этого можно приступать к использованию.

## Использование ##

В большинстве своем способ использования интуитивно понятен,
но есть пара особенностей

1. Яблоки тухнут через 5 минут, а не 5 часов. Сделано
это для облегчения тестирования. Механизм гниения
реализован через очереди, поэтому подмена таймштампа
не поможет. В качестве альтернативы можно было использовать
крон, но мне показалось, что для тестового задания
использовать очереди удобнее.
2. Кнопка генерации яблок генерирует от 1 до 10 яблок случайно
3. Дата создания проставляется рандомно в течение
последних суток
